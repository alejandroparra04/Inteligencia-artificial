Descripción del Código TensorFlow

import tensorflow as tf
import numpy as np

En esta parte se importa la biblioteca de tensor flow y la de numpy, la cual la primera permite construir y entrenar redes neuronales y la otra sirve para trabajar con arrays.

celsius = np.array([-40, -10, 0, 8, 15, 22, 38], dtype=float)
fahrenheit = np.array([-40, 14, 32, 46, 59, 72, 100], dtype=float)

Aquí se crea un array con los valores de temperatura en grados Celsius y Fahrenheit.

oculta1 = tf.keras.layers.Dense(units=3, input_shape=[1])
oculta2 = tf.keras.layers.Dense(units=3)
salida = tf.keras.layers.Dense(units=1)
modelo = tf.keras.Sequential([oculta1, oculta2, salida])

La primera linea muestra que tiene 3 neuronas y espera una entrada con una sola y la segunda toma los resultados de la primera, la salida sirve para predecir un valor y el modelo es un modelo secuancial.

modelo.compile(optimizer=tf.keras.optimizers.Adam(0.1), loss='mean_squared_error')

Aquí se configura el modelo Adam con una tasa de aprendizaje de 0.1.

historial = modelo.fit(celsius, fahrenheit, epochs=1000, verbose=False)

Entrenamos el modelo usando los datos de Celsius y Fahrenheitel cual se ajustara al modelo durante 1000 épocas.

import matplotlib.pyplot as plt
plt.plot(historial.history["loss"])

se importa el Matplotlib, y se genera un grafico para visualizar la evolucion.

resultado = modelo.predict([100.0])

se hace una predicción: aquí le pedimos al modelo que nos diga cuántos grados Fahrenheit corresponden a 100 grados Celsius.

print(oculta1.get_weights())

se imprimen los pesos internos de la primera capa para ver cómo el modelo ajustó a los valores durante el entrenamiento.

print(oculta2.get_weights())

Lo mismo, pero para la segunda capa oculta.

print(salida.get_weights())

Finalmente se imprimen los pesos de la salida, que es la que produce la predicción final.
